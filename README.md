# Bruno Recipes Site

Site unificado para publicar as paginas em caminhos do dominio principal:

- `/combo/`
- `/feijoada/`
- `/geleias/`
- `/molhos/`

Configuração recomendada no Cloudflare Pages:

- Build command: vazio
- Build output directory: `.`
- Custom domain: `brunorecipes.com`

Os checkouts da Hotmart continuam os mesmos dentro das paginas.
